## Arrays

- [CountPairsHavingSum](https://github.com/AmaanUllahKhan313/algorithms/blob/master/src/main/java/com/algos/arrays/CountPairsHavingSum.java) - Count pairs in an array with a given sum.
- [DifferenceBetweenLargest](https://github.com/AmaanUllahKhan313/algorithms/blob/master/src/main/java/com/algos/arrays/DifferenceBetweenLargest.java) - Difference between the largest and smallest elements.
- [FrequencyOfArrayElements](https://github.com/AmaanUllahKhan313/algorithms/blob/master/src/main/java/com/algos/arrays/FrequencyOfArrayElements.java) - Count frequency of each element in an array.
- [MaxProfitFromStocks](https://github.com/AmaanUllahKhan313/algorithms/blob/master/src/main/java/com/algos/arrays/MaxProfitFromStocks.java) - Maximize profit by choosing days to buy and sell stocks.
- [MajorityElement](https://github.com/AmaanUllahKhan313/algorithms/blob/master/src/main/java/com/algos/arrays/MajorityElement.java) - Find the element that appears more than n/2 times.
- [MaxOfJMinusI](https://github.com/AmaanUllahKhan313/algorithms/blob/master/src/main/java/com/algos/arrays/MaxOfJMinusI.java) - Maximum value of j - i such that arr[j] > arr[i].
- [MergeMPlusN](https://github.com/AmaanUllahKhan313/algorithms/blob/master/src/main/java/com/algos/arrays/MergeMPlusN.java) - Merge two sorted arrays with extra space in one.
- [PairInArrayDivisibleByK](https://github.com/AmaanUllahKhan313/algorithms/blob/master/src/main/java/com/algos/arrays/PairInArrayDivisibleByK.java) - Check if array can be divided into pairs divisible by k.
- [ReverseArray](https://github.com/AmaanUllahKhan313/algorithms/blob/master/src/main/java/com/algos/arrays/ReverseArray.java) - Reverse an array in place.
- [RotationUsingGCD](https://github.com/AmaanUllahKhan313/algorithms/blob/master/src/main/java/com/algos/arrays/RotationUsingGCD.java) - Rotate an array using the GCD method.
